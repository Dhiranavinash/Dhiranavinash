About Me

I am an Electronics & Telecommunication Engineer currently pursuing my MSc. at ITC, University of Twente.

Beyond datasets and algorithms, I'm a wildlife enthusiast who loves exploring nature with a camera in hand.

Projects

PFNO-SAR: A Physics-Informed Fourier Neural Operator for PolSAR De-speckling

Introduces PFNO-SAR, combining spectral and spatial convolutions with SAR-specific loss functions for structural consistency. Evaluated on ALOS-2 PALSAR L-band Mumbai data, it outperforms state-of-the-art methods in frequency-domain deep learning for PolSAR de-speckling.

Analysis of GAN-Based De-speckling for PolSAR Data

Custom GAN architecture (U-Net generator and CNN discriminator) for SAR de-speckling on AIRSAR Flevoland dataset. Outperforms traditional filters in speckle suppression and edge preservation, achieving EPI values above 0.90.

Land-Cover Classification Using ACO and PSO for Random Forest Optimization

Integrates ACO for feature selection with PSO for hyperparameter tuning in Random Forest classification. Using ALOS-2 PALSAR L-band data, the ACO-PSO-RF model achieves 13–15% accuracy improvement over unoptimized RF.

PolSARHub: A Comprehensive Benchmark Repository with Pre-trained U-Net Model

Labeled data repository and deep learning framework addressing ground truth scarcity in PolSAR classification. Provides multi-sensor datasets, pre-trained models, and a unified pipeline for training and benchmarking.

Comparative Analysis of Frequency-Domain Filters for Speckle Reduction

Evaluates nine frequency-domain filters for PolSAR speckle reduction. Extended Adaptive Wiener filter outperformed traditional Lee Refined in SNR, PSNR, and SSIM with minimal trade-offs.

Transfer Learning Based Approach for PolSAR Image Classification

User-friendly classification tool for ALOS2 PALSAR L-band imagery. Using Random Forest trained on Mumbai data, it classifies water, settlements, forests, wetlands, and mangroves with 65-95% accuracy across five regions.

SAR Image Classification using Probabilistic Adaptive Genetic Algorithm

PAGA optimizes training pixel selection for Random Forest classification of PolSAR data. Baseline RF accuracy of 64.31% increased to 85.41% with PAGA+RF, achieving higher accuracy while reducing training data requirements.
